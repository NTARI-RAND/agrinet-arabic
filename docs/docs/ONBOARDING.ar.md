# دليل الإعداد والتكامل

## مقدمة
يشرح هذا الدليل كيفية إعداد عقدة جديدة للاتصال بشبكة أغرينيت الموزعة.

## المتطلبات
- Python 3.8 أو أحدث
- - الوصول إلى خادم أغرينيت الرئيسي
  - - عنوان IP للعقدة الجديدة
   
    - ## خطوات الإعداد
   
    - ### 1. التثبيت الأساسي
    - ```bash
      pip install agrinet
      agrinet-cli init
      ```

      ### 2. إعدادات التكامل
      قم بتعديل ملف الإعدادات:
      ```yaml
      node:
        name: "my-node"
        public_ip: "192.168.1.100"
        port: 8000
      ```

      ### 3. الاتصال بالشبكة
      ```bash
      agrinet-cli connect --server http://main-server:8000
      ```

      ## التحقق من الإعداد
      ```bash
      agrinet-cli status
      ```

      ## الاستكشاف
      - تحقق من الحالة الصحية للعقدة
      - - تأكد من الاتصال بالعقد الأخرى
        - - راقب السجلات للأخطاء
         
          - ## الدعم
          - للحصول على المساعدة، راجع الوثائق الرسمية أو اتصل بفريق الدعم.
